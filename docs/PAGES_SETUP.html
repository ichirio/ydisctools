GitHub Pages Setup

Deployment notes for publishing docs/ with GitHub Actions.

Current setup

  • Workflow file: .github/workflows/pages.yaml
  • Publish source: GitHub Actions artifact from docs/

One-time repository setting

  1. Open repository settings on GitHub.
  2. Go to Settings > Pages.
  3. In Build and deployment, set Source to GitHub Actions.

Publish or update docs

  • Push any change under docs/ to main.
  • Or run the Deploy Pages workflow manually from Actions.

Expected site URL
